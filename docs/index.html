@webboot/magic-theme

this is the @magic-themes webboot theme. It is being used as the foundational theme for all @webboot pages.

installation

npm install --save --save-exact @webboot/magic-theme

usage

require in config.mjs:

// config.mjsexport default {  // ... other config  THEME: '@webboot/magic-theme',}

magic will then find and merge the themes automatically, just as it did on this page

theme vars

colors

this theme exports and uses the following colors.

  • vars.white: #ffffff
  • vars.neutral: #5a5a5a
  • background
    • vars.background.light: #e0e0e0
    • vars.background.dark: #212121
  • link
    • vars.link.light: rebeccapurple
    • vars.link.dark: rebeccapurple
    • link.hover
      • vars.link.hover.light: #412162
      • vars.link.hover.dark: #8f6ab4
  • primary
    • vars.primary.neutral: rebeccapurple
    • vars.primary.light: #8f6ab4
    • vars.primary.dark: #412162
  • text
    • vars.text.light: #212121
    • vars.text.dark: #c4c4c4

max page width

vars.maxWidth: 1000px

fade duration:

vars.fadeDuration should be used for all css animations

vars.fadeDuration: 500ms

breakpoints

the following width breakpoints exist

  • vars.widths.tablet: 500
  • vars.widths.laptop: 1024
  • vars.widths.desktop: 1600
